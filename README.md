# Escape of Cleopatra

Escape of Cleopatra is a 2D adventure/horror platformer game set inside an ancient Egyptian pyramid. Players take control of Cleopatra and must escape the pyramid by collecting sacred keys, unlocking doors, avoiding deadly traps, and surviving the relentless mummy that awakens during gameplay.

The game combines platforming, exploration, survival mechanics, and atmospheric horror elements to create a tense and immersive experience.

## Gameplay

- Control Cleopatra through dark pyramid chambers
- Collect sacred keys to unlock new areas
- Escape traps and dangerous obstacles
- Survive the mummy chase after picking up keys
- Manage torch light before darkness consumes you
- Progress through increasingly difficult levels

## Features

- 2D platformer gameplay
- Smart mummy AI chase system
- Dynamic torch-light mechanic
- Ancient Egyptian themed environments
- Atmospheric music and sound effects
- Story dialogues between levels
- Pause and audio toggle system
- Fullscreen WebGL support
- Keyboard-based controls

## How the Game Was Built

The project was developed step-by-step starting from a simple Unity prototype.

### Development Process

1. Created the initial game prototype in Unity
2. Implemented player movement and level mechanics
3. Added key collection and door progression system
4. Developed the mummy AI chase behavior
5. Designed the torch-light survival mechanic
6. Added sound effects, music, and UI elements
7. Designed multiple levels with increasing difficulty
8. Optimized the game for WebGL browser deployment

### Core Systems Implemented

- Character movement and jumping
- Enemy AI path chasing
- Torch light reduction system
- Collision and trap handling
- UI and pause menu systems
- Audio management
- Level progression logic
